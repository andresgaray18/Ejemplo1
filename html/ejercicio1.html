What is XML?

  • XML stands for EXtemsibleMarkup Language
  • XML is a markup language much like HTML
  • XML was designed to describe data
  • XML tags are not predefined. You must define your own tags
  • XML uses a Document Type Definition(DTD) or an XML Shema to describe
    the data
  • XML with a DTD or XML Schema is designed to be self-descriptive

The main difference between XML and HTML

XML was designed to carry data.

XML is not a replacement for HTML.

XML and HTML were designed with different goals:

XML was designed to describe data and to focus on what data is.

HTML was desingned to display data and to focus on how data looks.

HTML is about displaying informatio, while XML is about describing information.

XML does not DO anything

XML was not designed to DO anything.

Maybe it is a little herd to understand, but XML does not DO anything. XML was created
